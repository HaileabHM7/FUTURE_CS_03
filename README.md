# Secure File Sharing System

This project implements a secure file upload and download system using AES encryption.

## Features
- AES-256 encryption for files at rest
- Secure upload and download
- Flask backend
- Simple web interface

## Technologies
- Python
- Flask
- PyCryptodome
- AES (GCM mode)

## How to Run
1. Install dependencies
2. Set AES_SECRET_KEY in .env
3. Run `python app.py`
